# Admin Web

Vue 3 admin frontend scaffold for the publishing management console.

## Stack

- Vue 3
- Vite
- TypeScript
- Pinia
- Vue Router
- Element Plus

## Notes

- This scaffold follows the thin-shell direction discussed for `pure-admin-thin`.
- It intentionally keeps logic simple and moves most structure into reusable page shells and typed route metadata.
- Dependencies are declared but not installed in the current environment.
